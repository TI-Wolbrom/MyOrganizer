Opis programu:

	  MyOrganizer to program użytkowy, który pozwala ci na zaplanowanie swojego idealnego dnia.
	  Program zawiera następujące moduły:
	  
	  Moduł planu lekcji:
	  	- Pozwala na stworzenie i zarządzanie twoim planem lekcji.
	  
	  Moduł wydarzenia:
	  	- Pozwala na ustawienie powiadomień o wybranych terminach.
	  	- Dostępne są 3 typy zadań:
	  		- Komunikat
	  		- Wyłącz komputer
	  		- Wyloguj
	  
	  Moduł siłowni:
	  	- Pozwala na dodanie maksymalnie 15 planów treningowych.
	  	- Zawiera kalkulatory spalania kalorii, LBM oraz BMI.
	  	
	  Nasz program zadba również abyś nie zapomniał o ważnych wydarzeniach.
	  Program wspiera trzy języki: Polski, Angielski oraz Niemiecki.
	  Dzięki systemowi aktualiazcji twoja kopia programu będzie zawsze aktualna.
	  Jeżeli chciałbyś wyłączyć aktualizacje możesz zawsze to zrobić w ustawieniach programu co jednak odradzamy.
	  
Wymagania sprzętowe:

    - Procesor posiadający taktowanie co najmniej 1GHz
    - Pamięć RAM: 256MB
    - Wolne miejsce na dysku twardym: ~1MB
    - Zainstalowana Java 8 (zalecane Version 8 Update 161 lub nowsza)
    - Dostęp do internetu w razie automatycznych aktualizacji.
    - Wspierane systemy: 
        - Windows (100%), 
        - Linux (~80%),
    - Nie testowano na:
        - Mac OSX (? %)

Jeżeli otrzymujemy komunikat typu:

	- "Jedna instancja jest dozwolona" natomiast program jest wyłączony zalecamy uruchomić skrypt fixlock.bat (Windows) lub fixlock.sh (Linux) w zależności od systemu.
